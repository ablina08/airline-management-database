# Airline Management System Database

## Project Description
This project represents a database system for managing airline operations including flights, passengers, bookings, pilots and payments.

## Technologies Used
- PostgreSQL
- pgAdmin
- SQL
- GitHub

## Database Structure
The system includes the following tables:

- Airlines
- Airports
- Aircrafts
- Pilots
- Passengers
- Flights
- Bookings
- FlightPilots
- Luggage
- Payments

## Features
- Foreign Key relationships
- Constraints
- Triggers
- Sample data
- Analytical queries

## How to Run
1. Open pgAdmin
2. Run airline_schema.sql
3. Run sample_data.sql
4. Run triggers.sql
5. Test queries.sql
